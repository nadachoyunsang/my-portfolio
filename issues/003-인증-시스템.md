# #3 인증 시스템 (Google OAuth)

- **라벨**: feature, security
- **상태**: done
- **의존성**: #2

## 설명

Supabase Auth를 활용한 Google OAuth 인증을 구현합니다.
관리자 페이지는 지정된 구글 계정 1개만 접근 가능하며, 그 외에는 404 페이지로 리디렉션합니다.

## 작업 목록

- [x] 인증 유틸 함수 (`getUser`, `getAdminUser`)
- [x] OAuth 콜백 Route Handler (`/auth/callback`)
- [x] 관리자 레이아웃 및 대시보드 (`/admin`)
- [x] 로그인 페이지 (`/auth/login`) — Google OAuth
- [x] 로그아웃 버튼 (`LogoutButton`)
- [x] 미인증/미인가 사용자 404 처리 (`notFound()`)
- [ ] Supabase Auth에 Google OAuth Provider 설정 (대시보드 수동 작업)
- [ ] 허용 관리자 이메일 환경변수 등록 (`ADMIN_EMAIL`)

## 보안 요구사항

- 관리자 경로: `/admin/*`
- 미인증 접근 → 404 페이지 (관리자 페이지 존재 자체를 노출하지 않음)
- 인증되었으나 허용 계정이 아닌 경우 → 404 페이지
- 허용 계정: 환경변수 `ADMIN_EMAIL`로 관리
