# #7 관리자 - 블로그 글 관리 (CRUD)

- **라벨**: feature, admin
- **상태**: open
- **의존성**: #3, #5

## 설명

관리자 페이지에서 블로그 글을 작성, 수정, 삭제할 수 있는 기능을 구현합니다.
비개발자도 쉽게 사용할 수 있는 WYSIWYG 편집기(Tiptap)를 적용합니다.

## 작업 목록

- [ ] 관리자 블로그 글 목록 페이지 (/admin/posts)
- [ ] 글 작성 페이지 (/admin/posts/new)
- [ ] 글 수정 페이지 (/admin/posts/[id]/edit)
- [ ] Tiptap WYSIWYG 편집기 구성
  - [ ] 기본 서식 (굵게, 기울임, 제목, 목록)
  - [ ] 이미지 삽입 (Supabase Storage 업로드)
  - [ ] YouTube 영상 임베드
- [ ] 카테고리 선택 (다큐멘터리/책/기사)
- [ ] 태그 입력
- [ ] 썸네일 이미지 업로드
- [ ] 요약(excerpt) 입력
- [ ] 공개/비공개 토글
- [ ] 글 삭제 (확인 다이얼로그)
- [ ] 이미지 업로드 시 자동 리사이징

## 참고

- 이미지 리사이징: 업로드 시 클라이언트에서 최대 1200px 너비로 압축
- 썸네일: 최대 600px 너비
- slug 자동 생성 (제목 기반, 수동 수정 가능)
