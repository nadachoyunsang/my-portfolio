# #8 관리자 - 사이트 콘텐츠 관리

- **라벨**: feature, admin
- **상태**: open
- **의존성**: #3, #4

## 설명

관리자 페이지에서 홈 화면의 자기소개, 연락처, 대표 콘텐츠를 수정할 수 있는 기능을 구현합니다.

## 작업 목록

### 자기소개 관리 (/admin/profile)

- [ ] 이름 수정
- [ ] 직업 수정
- [ ] 소개글 수정
- [ ] 프로필 사진 업로드

### 연락처 관리 (/admin/contact)

- [ ] 이메일 주소 수정
- [ ] SNS 링크 추가/수정/삭제 (GitHub, LinkedIn 등)

### 대표 콘텐츠 관리 (/admin/featured)

- [ ] 대표 콘텐츠 추가 (제목, 설명, 썸네일, 링크)
- [ ] 대표 콘텐츠 수정
- [ ] 대표 콘텐츠 삭제
- [ ] 드래그 또는 버튼으로 순서 변경

## 참고

- site_content 테이블의 key-value 구조로 관리
- 대표 콘텐츠는 featured_content 테이블로 관리
- 변경 즉시 반영 (ISR revalidation)
