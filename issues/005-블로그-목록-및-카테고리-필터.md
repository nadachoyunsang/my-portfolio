# #5 블로그 목록 및 카테고리 필터

- **라벨**: feature
- **상태**: open
- **의존성**: #2, #4

## 설명

홈 페이지 내 블로그 섹션에 리스트형 글 목록과 카테고리 탭 필터를 구현합니다.

## 작업 목록

- [ ] 블로그 섹션 영역 구현
- [ ] 카테고리 탭 필터 UI (전체 / 다큐멘터리 / 책 / 기사)
- [ ] 리스트형 글 목록 (썸네일 + 제목 + 요약 + 태그)
- [ ] Supabase에서 글 목록 조회 (카테고리 필터링)
- [ ] 공개(published)된 글만 표시
- [ ] 페이지네이션 또는 더보기 버튼
- [ ] 글 없을 때 빈 상태 UI

## UI 구조

```
다큐멘터리  │  책  │  기사  │  전체     ← 탭 필터
──────────────────────────────────

│ [썸네일]  제목                  │
│           요약 텍스트...        │  ← 리스트 아이템
│           #태그1 #태그2        │
└──────────────────────────────┘
```
