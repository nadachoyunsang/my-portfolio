# #6 블로그 글 상세 페이지

- **라벨**: feature
- **상태**: open
- **의존성**: #5

## 설명

블로그 글 목록에서 클릭 시 이동하는 개별 글 상세 페이지를 구현합니다.

## 작업 목록

- [ ] 동적 라우트 구현 (/blog/[slug])
- [ ] 글 상세 조회 (Supabase)
- [ ] HTML 본문 렌더링 (Tiptap 출력물)
- [ ] 썸네일 이미지 표시
- [ ] 카테고리, 태그, 작성일 표시
- [ ] YouTube 영상 임베드 지원
- [ ] 목록으로 돌아가기 링크
- [ ] 미공개 글 접근 시 404 처리
- [ ] SEO 메타태그 (title, description, OG)

## 참고

- 정적 생성(SSG) + ISR(Incremental Static Regeneration) 적용으로 로딩 속도 최적화
- slug 기반 URL 구조: `/blog/my-first-post`
