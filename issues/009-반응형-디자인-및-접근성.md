# #9 반응형 디자인 및 접근성

- **라벨**: enhancement, a11y
- **상태**: open
- **의존성**: #4, #5, #6

## 설명

모든 페이지에 반응형 디자인을 적용하고, WCAG 2.1 AA 표준을 준수합니다.
주요 브라우저(Chrome, Safari, Edge) 및 모바일 브라우저에서 정상 작동을 확인합니다.

## 작업 목록

### 반응형 디자인

- [ ] 브레이크포인트 정의 (모바일 < 768px, 태블릿 < 1024px, 데스크톱)
- [ ] 헤더 모바일 메뉴 (햄버거 메뉴)
- [ ] 블로그 리스트 모바일 레이아웃 (세로 배치)
- [ ] 글 상세 페이지 모바일 최적화
- [ ] 관리자 페이지 모바일 대응 (최소한)
- [ ] 이미지 반응형 처리 (Next.js Image)

### 접근성 (WCAG 2.1 AA)

- [ ] 색상 대비 비율 4.5:1 이상 확인
- [ ] 키보드 네비게이션 지원
- [ ] 시맨틱 HTML 태그 사용 (header, nav, main, article, footer)
- [ ] 이미지 alt 텍스트
- [ ] 포커스 표시 스타일
- [ ] aria-label 적용 (아이콘 버튼 등)
- [ ] 스크린 리더 테스트

### 크로스 브라우저

- [ ] Chrome 테스트
- [ ] Safari 테스트
- [ ] Edge 테스트
- [ ] 모바일 Chrome/Safari 테스트
