# #4 홈 페이지 레이아웃 및 공통 UI

- **라벨**: feature, ui
- **상태**: open
- **의존성**: #1

## 설명

사이트 전체 레이아웃과 홈 페이지의 자기소개/연락처 섹션을 구현합니다.
어둡고 세련된 다크 테마를 적용합니다.

## 작업 목록

- [ ] 전체 레이아웃 구성 (헤더, 메인, 푸터)
- [ ] 헤더: 이름/소개글 + 섹션 앵커 메뉴 링크
- [ ] 자기소개 섹션: 이름, 직업, 소개글
- [ ] 푸터: 연락처 (이메일, SNS 링크)
- [ ] 다크 테마 색상 체계 적용
- [ ] 공통 컴포넌트 (버튼, 카드, 구분선 등)
- [ ] 스크롤 시 메뉴 활성 상태 표시

## 디자인 사양

- **배경**: 어두운 색 (#0a0a0a ~ #111)
- **텍스트**: 밝은 색 (#f0f0f0, #ccc)
- **강조색**: 추후 결정 (기본 블루 계열)
- **레이아웃**: 최대 너비 제한, 중앙 정렬
- **헤더**: 상단 고정, 스크롤 시 배경 불투명
- **섹션 전환**: 부드러운 스크롤 (scroll-behavior: smooth)
